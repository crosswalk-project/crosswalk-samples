This sample demonstrates the CSS3 media features for adapting the same content to different output devices and screens.

This content display in right color if test device satisfy the CSS3 Media Queries.

The content can be set to green color if the device aspect ratio is less than or equal to 1/2

Set green

The content can be set to red color if the device aspect ratio is more than or equal to 1/10

Set red

The content can be set to blue color if the device height is less than or equal to 1300px

Set blue

The content can be set to yellow color if the device height is more than or equal to 100px

Set yellow

It is designed by device-aspect-ratio and device-height properties of CSS @media
