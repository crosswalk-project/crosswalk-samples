This sample demonstrates the NFC with read, write and send operation.

When running this sample, moving NFC tag close to the device to detect the NFC connection.

Turn OFF NFC Turn ON NFC
Tap to Read NDEF Tap to Write text NDEF Tap to Write URI NDEF Tap to Write Media NDEF
Send URI NDEF to peer

Note: Please confirm NFC is truned on before doing read or wirte operation.

This sample is designed by nfc interface.
