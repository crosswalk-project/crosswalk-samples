This sample demonstrates the device is able to handle pageVisible event when lock and unlock the screen by gesture.

1.Press power key to lock the screen, then press power key again to unlock the screen.

2.Click "Screen-lock State" to check if the visibilitystate attribute is set to hidden and the hidden attribute is set to true when lock the screen.

Screen-lock State
