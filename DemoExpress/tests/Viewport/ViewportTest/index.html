1. The width and height of page should be same as the device, and not scale more smaller after set with "width=device-width, height=device-height"

2. The page will be scaled to bigger than the initial page, and not scalable after set with "initial-scale=2.0, user-scalable=no"

3. The page is scalable from 100% to 300% if device support the scale after set with "minimum-scale=1.0, maximum-scale=3.0, user-scalable=yes"
