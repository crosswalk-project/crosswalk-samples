This sample demonstrates viewport with functionality of setting device height and width, scale, enabling or disabling user scalable to allow or disallow zoom in/out screen, sliding the screen by figure.

  1. Launch sub app "ViewportTest"
    • On IVI:
      1. Execute command "xwalkctl" with app user to get "ViewportTest" app id
      2. Launch "ViewportTest": xwalk-launcher <app-id>
  2. Click the button in "ViewportTest" to test Viewport

This sample is designed by width, height, initial-scale, minimum-scale, maximum-scale and user-scalable property.
